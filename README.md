## Features

1. Crop Image
2. Add Border to image
3. Convert to Black and White
4. Resize Image
5. Convert to PNG or JPG vica-versa
6. Add Watermark on the image
7. Rotate image
8. Flip Image
9. Draw text on image

### System Requirements
1. Linux (Tested), Windows
2. Python 3.6 + (PIL library from Image)
